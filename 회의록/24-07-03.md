## 3. 커밋 컨벤션 재정의

-   feature : 코드 수정 (로직 변경 및 변수 추가, 수정) -> 결과값(type)이 변함
-   refactor : 코드 수정 -> 결과값은 변하지 않음
-   rename : 변수, 함수, 디렉토리, 파일의 이름만 변경
